Fresh Installation
  1. Copy all Files from the Directory phpBB2 in the Package to a directory on your web server, for example /phpBB2/
  2. Run the Installer, http://my.forums.com/phpBB2/install/install.php replacing my.forums.com/phpBB2/ with the path of your forum, using your web browser.
  3. Enter all Data and finish the installation.
  4. Delete or rename the directory /install.
  5. Change the permissions for the Directories and the files like "File- and directorypermissons on *NIX-Server"
  6. Use the new Forum ;)

Upgrade from existing phpBB2 Plus 1.52
  1. It is important that you follow these steps exactly because you may get in troubles if you don't !

  2. Backup all your Data ! All files from your FTP and your SQL Database ! Don't cry if something goes wrong and you have no backup !
  3. You will need the Data of the following directories later in the new Forum so please BACK them up NOW!! : /album_mod/upload, album_mod/upload/cache, images/avatars, images/smiles, files, images/avatars/gallery, images/ranks, /pafiledb/images/screenshots, /pafiledb/uploads !!!
  4. Copy ONLY the directory update/ to your existing phpBB2 Plus 1.52 root directory.
  5. Log into your phpBB2 Plus 1.52 forum as Administrator
  6. Run the index file in your Browser (http://my.forums.com/phpBB2/update/index.php). Start all available updates step by step, this may take some time so don't cancel after running !
  7. Download the file config.php from your existing phpBB2 Forum and put it in a secure place (You will need it later !)
  8. Then delete all Files from your old Forum on the Webspace (You made the Backup, right ? if not, this is your last chance to back them up). If you backed them up then go ahead and delete ALL Files !
  9. Copy all Files from the phpBB2 dir to your Webspace.
  10. Copy the file config.php that you backed up to your phpBB2 Dir on the Webserver overwrite the existing one.
  11. Copy the files from the directories /album_mod/upload, album_mod/upload/cache, images/avatars, images/smiles, files, images/avatars/gallery from your old forum back into the directories of your new forum, so your old files, avatars and album pics will be back.
  12. The ranks images from images/ranks must be uploaded in templates/fisubsilversh/images/lang_***/ranks. Do this for all installed templates and languages as well.
  13. Delete the file update/ directory
  14. Change the permissions for the directories and the files like "File- and directorypermissons on *NIX-Server"
  15. Your are done, you should have version 1.53 based on phpBB 2.0.21 now.

Upgrade from existing phpBB 2.0.XX
  1. If you are already using Mods in your Forums that required Database Changes, you may have problems installing this version. The included Database Update tool only creates fresh tables for the Mods and does NOT update any existing MODs you are using to the latest version. So what does that mean ? If you are running a phpBB 2.0.8-2.0.21 with for example SmartorAlbum 2.0.42 or Lower you first have to update your Database to the Album v2.0.53 that is included in phpBB2 plus. Check the Mods List at the beginning of this readme file for information on the versions that are used in phpBB2 plus. If you have no Mods in your Forum that required Database Changes, then just ignore this and go on to step 2.
  2. Backup all your Data ! All Files from your FTP and also your SQL Database ! Don't cry if something goes wrong and you have no Backup !
  3. You will need the Data of the following directories later in the new Forum so please BACK them up NOW!! : images/avatars, images/smiles, files, images/avatars/gallery, images/ranks!!!
  4. Unzip the phpBB2 plus zip file to a directory on your local hard disk.
  5. Only upload the directory update/ to your existing phpBB2 Directory on the webserver, do NOT COPY THE OTHER FILES !!!!
  6. Login to your Forum as Administrator
  7. Using your web browser, open the index.php File you uploaded (http://your.forums.com/phpBB2/update/index.php) and run all available updates.
  8. You should see additional tables are created or updated (Added/Updated)
  9. Download the file config.php from your existing phpBB2 Forum and put it in a secure place (You need it later !)
  10. If you used the attachment mod before, you should backup your /files directory now. Also backup the users avatar directory which is images/avatars to use them later again.
  11. Did you Backup your old Data already? If not, now it's the last chance to do it !
  12. Delete all Files from your old forums now. If you have  your own templates, images, or graphics, then back them up as well)
  13. Copy all Files from the phpBB2 directory of the phpBB2 plus 1.53 package to your Webserver (including directory structure)
  14. Copy all ranks from your old images/ranks up to templates/fisubsilversh/images/lang_***/ranks. Do this for all installed templates and languages as well.
  15. Copy the file config.php that you backed up before to your phpBB2 Dir on the Webserver overwriting the existing one)
  16. Delete or rename the directory /install
  17. Change the ermissions for the directories and the files like "File- and directorypermissons on *NIX-Server"
  18. Upload the "old" avatars and maybe files directory to your new Forum
  19. Use the new Forum ;)

Upgrade from existing phpBB2 Plus 1.51 or lower

 phpBB2.de will not support phpBB2 Plus versions lower then phpBB2 Plus 1.52. If you want to use phpBB2 plus 1.53 you first have to upgrade your Plus Forum to a Plus 1.52 which can be found at www.phpbb2.de. Download the package and follow the instructions in the included /docs/install.html. If the upgrade to phpBB2 Plus 1.52 was successful and your Forum is running you can do the installation of phpBB2 plus as described In the last Chapter

Settings to do after installation

Go to Admin Panel into Plus -> Configuration and set the Contact EMail Adress to the one you wish to receive the EMails that are sent through the Contact Form on your Site. If you don't change this value to a valid address you will never receive any EMail through your Contact Form.

Go to Admin Panel into Group Admin -> Color Groups and set up your Color Groups and Members. For example if you are the Board Administrator you should put your Username to the Administrators Group otherwise your Color in the List of Online Users will look as a normal User.

    Frequently Asked Questions (FAQ)

    Can i use my own template?
    Yes you can, but all Mods that are used in phpBB2 plus require some changes in your template files to be compatible with Plus 1.53. You can have a look at the included Mods List which can be found at the beginning of this document. You need these Mods and the installationl instructions included in them to make the template files (*.tpl or *cfg) changes. DO NOT make any changes to the *.php Files !!! Only change the Template Files !!! If you did everything successfully, you can submit you new template to phpbb2.de so that other Users could benefit from it.

    Why isn't MOD x or y in this package?
    Because phpBB2.de does not want to overload this package. The included Mods will provide a lot of additional functions that are not in phpBB 2.0.21 Standard edition. If you want to use other Mods you can suggest a Mod in our Forums, maybe it will be added to a future release.

    I need another Language than German or English, what can I do?
    If you need another language you have to install the standard Language File of your Country's language from www.phpbb.com and make the additional changes in the language files. Open docs/mods and look through all MODs installation instructions and make the changes for the language files. If you want to provide your successfully finished language package to  phpbb.de, then please submit them at www.phpbb2.de
